Evolution of the Web

  • The history of the web we know today can be categorized into three phases - Web 1.0, Web 2.0, and Web 3.0
    • Web 1.0
      • ~1995
      • "read-only web" - websites were informational, static, and were linked together by hyperlinks
      • text-only e-mails could be written and sent
      • "content delivery network (CDN)" - users passively received the information shown on websites without interacting with it
